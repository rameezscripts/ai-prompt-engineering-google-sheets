Step 01

Technique: Few-Shot Prompting

Show Pattern → AI Copies It

Instead of explaining what you want, show AI 2 examples. It reads the pattern and applies it to new items. Perfect for generating repetitive CRUD functions.

Few-Shot: CRUD Generator Gemini 
Generate Apps Script CRUD functions following this exact pattern:

Example 1 — Sheet: "Employees"
function getEmployees() {
  const sh = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Employees');
  const data = sh.getDataRange().getValues();
  const headers = data.shift();
  return data.filter(r => r[0] !== '').map(r => Object.fromEntries(headers.map((h,i) => [h,r[i]])));
}
function addEmployee(obj) {
  const sh = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Employees');
  const headers = sh.getRange(1,1,1,sh.getLastColumn()).getValues()[0];
  sh.appendRow(headers.map(h => obj[h] || ''));
  return {success: true, msg: 'Employee added'};
}

Example 2 — Sheet: "Products"
function getProducts() {
  const sh = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Products');
  const data = sh.getDataRange().getValues();
  const headers = data.shift();
  return data.filter(r => r[0] !== '').map(r => Object.fromEntries(headers.map((h,i) => [h,r[i]])));
}
function addProduct(obj) {
  const sh = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Products');
  const headers = sh.getRange(1,1,1,sh.getLastColumn()).getValues()[0];
  sh.appendRow(headers.map(h => obj[h] || ''));
  return {success: true, msg: 'Product added'};
}

Now generate the same for: "Departments", "Attendance", "Invoices"
Few-Shot: Sheets Formula Pattern ChatGPT 
Generate Sheets VLOOKUP formulas following this pattern:

Example:
  Need: employee name from ID → =IFERROR(VLOOKUP(A2, Employees!A:B, 2, FALSE), "Not Found")
  Need: product price from code → =IFERROR(VLOOKUP(B2, Products!A:C, 3, FALSE), "Not Found")

Now generate for:
  Need: dept name from dept ID (Departments: A=ID, B=Name)
  Need: student grade from roll number (Results: A=Roll, D=Grade)
  Need: supplier phone from name (Suppliers: A=Name, C=Phone)
Why it works: AI doesn't just copy — it understands your naming convention, code style, and return format, then applies all of it to new items.
Step 02

Technique: Role Prompting

Act Like a Senior Developer

Tell AI "You are a senior Apps Script developer with 10 years experience." AI immediately writes more professional, optimized code — like asking a specialist instead of a junior.

Role: Fix Slow Apps Script Code Gemini 
You are a senior Google Apps Script developer with 10 years
of experience. You care deeply about performance.

This code processes 5000+ rows and is very slow:

```javascript
function processOrders() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Orders");
  var lastRow = sheet.getLastRow();
  for (var i = 2; i <= lastRow; i++) {
    var status = sheet.getRange(i, 5).getValue();
    if (status === "Pending") {
      sheet.getRange(i, 6).setValue("Processing");
      var email = sheet.getRange(i, 3).getValue();
      var name = sheet.getRange(i, 2).getValue();
      GmailApp.sendEmail(email, "Order Update", "Dear " + name + ", your order is processing.");
      sheet.getRange(i, 5).setValue("Processed");
    }
  }
}
```

As a senior developer:
1. Why is this slow? (simple explanation)
2. Rewrite using getValues()/setValues() batch operations
3. Any other improvements you recommend?
Role: Sheets Formula Expert DeepSeek 
You are a Google Sheets formula expert specializing in QUERY and ARRAYFORMULA.

Sheet "Sales": A=Date, B=Salesperson, C=Region, D=Product, E=Amount, F=Status

I need ONE formula in H2 that:
- Filters Status = "Completed" only
- Groups by Salesperson
- Sums total Amount per person
- Sorts highest to lowest
- Shows top 10 only

Give me the formula + simple explanation of each part.
Step 03

Technique: Chain-of-Thought

Make AI Think Step by Step

For complex problems, tell AI to think through it step by step before writing code. This forces careful reasoning instead of jumping to a wrong answer. Especially useful for debugging.

CoT: Debug a Daily Trigger Claude 
<context>
Apps Script time-based trigger runs every day at 8 AM.
It stopped working 3 days ago. No errors in execution log.
</context>

<code>
function sendAttendanceReminder() {
  var ss = SpreadsheetApp.openById('abc123xyz');
  var sheet = ss.getSheetByName('Employees');
  var data = sheet.getDataRange().getValues();
  for (var i = 1; i < data.length; i++) {
    if (data[i][4] === 'Active') {
      GmailApp.sendEmail(data[i][2], 'Attendance Reminder',
        'Please mark attendance: ' + new Date());
    }
  }
}
</code>

<task>
Think step by step:
Step 1: Trigger issues — deleted? expired? hit quota?
Step 2: Auth issues — scopes changed? re-auth needed?
Step 3: Gmail quota — daily sending limit reached?
Step 4: Sheet issues — renamed? permission changed?
Step 5: Code logic — any edge case that fails silently?

For each step: most likely cause → how to verify → how to fix.
</task>
CoT: Design a Formula DeepSeek 
I need an attendance percentage formula. Think through the logic first:

Sheet "Attendance": A=Date, B=EmployeeID, C=Status
Status values: "Present", "Absent", "Leave", "Half Day"
Rules: Half Day = 0.5, Leave does NOT count in total days
Formula for E2, employee ID is in D2.

Think step by step:
1. How to count total rows for this employee?
2. How to exclude "Leave" from total?
3. How to count "Present" days?
4. How to handle "Half Day" as 0.5?
5. What is the percentage formula?
6. How to handle divide-by-zero?

Then: give the final formula for E2 + simple explanation.
Don't just say "think step by step" — give the actual steps you want AI to think about. Vague instructions = vague thinking.
Step 04

Technique: System Prompts

Set Rules Once → Apply to Every Message

Set your stack, coding rules, and return format once — AI follows them for the entire conversation. No more repeating yourself every message.

In ChatGPT: Custom Instructions. In Gemini: System Instructions. In Claude: paste at top of first message.

System Prompt: Apps Script Assistant ChatGPT 
You are my Google Apps Script coding assistant.

Stack:
- Google Apps Script (V8 runtime)
- Google Sheets as database
- HTML Service for frontend
- No external libraries

Rules for ALL responses:
1. Always const/let — never var
2. Batch reads: getValues() once → process in memory → setValues() once
3. Never read/write cells inside loops
4. Use map/filter/reduce over for loops
5. Return format: {success: true/false, data: ..., msg: "..."}
6. Short developer-style comments only — no tutorial comments
7. Use GmailApp for emails, not MailApp
8. Wrap DB operations in try/catch
9. When I give column map, use those exact indexes

Output: code only unless I ask for explanation. ```javascript blocks.
System Prompt: Sheets + Script Expert Gemini 
You are my Google Sheets and Apps Script expert.

My context:
- Dashboards and automation using Sheets as DB
- Frontend: HTML Service + React 18 (CDN)
- Deploy as web apps via doGet()
- Non-technical business users

Coding standards:
- V8 runtime — modern JS only
- Batch operations always (getValues/setValues)
- SWR caching with CacheService where needed
- All functions return: {success, data, msg}
- Short comments like a developer's personal notes

For formulas:
- ARRAYFORMULA for auto-fill
- IFERROR for missing data
- QUERY over nested IFs for filtering

Output: code/formula only. No filler text. No markdown prose.
Step 05

Technique: Iterative Refinement

Build Layer by Layer — Not All at Once

Don't ask for a complete feature in one shot. Build it in layers — each prompt adds one piece. You stay in control and understand every line.

1

Core Function

"Write a function that reads today's rows from the Sales sheet and returns them as an array."

2

Add Email Logic

"Now write a function that takes that array and sends a summary email to manager@company.com — list each sale with name and amount, total at the bottom."

3

Add Error Handling

"Update: if array is empty, send a 'No sales today' email. Wrap in try/catch. Log errors to a 'Logs' sheet."

4

Add Trigger

"Write a function that creates a daily trigger to run this at 6 PM. Delete old trigger first to avoid duplicates."

5

Final Review

"Review all the code as a senior Apps Script developer. Check for quota issues, performance problems, anything that could fail. Fix what you find."

Apps Script Prompt Cheat Sheet

Copy these templates, fill in your details, and use them every time.

New Function

Bug Fix

My function should [expected] but it's [actual behavior]. [paste code] Cols: A=x, B=y, C=z Expected: [correct result] Actual: [wrong result] Find bug. 1 line explanation. Show fixed code only.

Performance Fix

You are a senior Apps Script dev. This code is slow — [X] rows, takes [Y] seconds. [paste code] Rewrite using: - getValues() batch read - setValues() batch write - No reads/writes in loops

Sheets Formula

Write a Sheets formula for [cell]. Sheet "[name]" columns: A=x, B=y, C=z Need: - [what to calculate] - [any filters or conditions] - ARRAYFORMULA for auto-fill Formula only. No explanation.

Add a Feature

Update this function to also [new feature]. [paste code] New requirement: - [what to add] - [condition] - Don't break existing logic Same code style. Code only.

Explain Code

Explain this Apps Script code in simple words. [paste code] Tell me: 1. What does it do overall? 2. What does each part do? 3. Any risks or edge cases? Simple language. No jargon.
